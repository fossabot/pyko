### Installing in your OS: 
1. Update PIP in python3.x 
> python3 -m pip install --upgrade
2. Install requeriments with file text
> python3 -m pip install -r requeriments.txt

### Work with Pyko:
* use all modules
```
from pyko import *
```
* module specific
```
from dev.colors import *
from dev.mm import *
```
